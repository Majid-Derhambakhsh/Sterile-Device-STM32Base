# Sterile Device STM32Base
Sterile system based on STM32F103C8T microcontroller

### Version : 1.0.0

- #### Type : Open Source ~ Embedded Device.

- #### Processor : STM32F103C8T6.

- #### Program Language : C

- #### Compiler : IAR Embedded Workbench.

### PCB Properties :

- #### Type : Single Layer

- #### Size : 40mm * 40mm

- #### Min Trace width : 0.2mm
